В очереди 1 канал. Лимит на количество ожидающий не назначен.
Пока канал занят, остальные ждут.

  - Поступает запрос.
  - есть свободный канал.
  - запрос начинает обрабатываться:
    - считывается файл с диска
    - передается в dtbase
    - dtbase возвращает результат
  - для select:
    - в ответ возвращается результат выборки - массив с данными
  - для insert, update, delete:
    - при успехе возвращается 'ok'.
    - результат записывается в файл на диск
  - канал освобождается для следующего запроса
