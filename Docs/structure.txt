dtqueue:
  - вход в приложение
  - создает очередь доступа к базе
  - читает данные с диска и передает в базу
  - получает результат из базы и записывает на диск

dtbase:
  - запускает итерацию данных и отдает результат
  - работает с методами select, insert, update, delete

dtlib:
  - подгружает методы из разрешенных в файле каталогов:
    - dtmethods
    - libs

dtreadfile:
  - читает файл с диска

dtwritefile:
  - пишет файл на диск

concurrentQueueThenable:
  - class Queue - асинхронная очередь
  - thenable
